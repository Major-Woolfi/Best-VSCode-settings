# VS Code Settings — Топовые настройки для разработчика

**Best-VSCode-settings** — набор тщательно подобранных настроек и шрифта Monocraft для максимально комфортной работы в Visual Studio Code.

Это не просто конфигурация — это готовое рабочее окружение, оптимизированное для продуктивности и эстетического комфорта.

![Status](https://img.shields.io/badge/Status-Active-brightgreen.svg)

## 🎯 Цели проекта

- Предоставить готовый набор настроек для VS Code
- Внедрить моноширинный шрифт Monocraft для комфортного чтения кода
- Создать базу для кастомизации под свои нужды

## 📁 Архитектура

```plaintext
├── settings.json            # Полная конфигурация редактора (UI, темы, плагины, форматирование)
├── Monocraft.ttf            # Моноширинный шрифт для VS Code
├── LICENSE                  # MIT License
└── README.md                # Документация по установке и использованию
```

## 📋 Быстрый старт

### Установка шрифта

1. Откройте файл `Monocraft.ttf`
2. Нажмите «Установить» в открывшемся окне
3. Перезапустите VS Code

### Применение настроек

1. Откройте VS Code
2. Нажмите `Ctrl+Shift+P` → введите `Open Settings (JSON)`
3. Замените содержимое файла на содержимое `settings.json` из репозитория
4. Сохраните файл (`Ctrl+S`)

> ⚠️ **Важно:** сделайте резервную копию текущих настроек перед заменой.

## 📌 Рекомендации

- **Резервное копирование:** перед заменой настроек сохраните текущий `settings.json`
- **Совместимость:** некоторые настройки требуют установленных расширений VS Code
- **Кастомизация:** этот набор — отправная точка. Настраивайте под себя.
